Screensaver is a lab information management system (LIMS) for high-throughput screening (HTS) facilities that perform screens of small molecule and RNAi libraries. The application has been developed by the ICCB-Longwood/ NSRB facility at Harvard Medical School.

Screensaver is an open source software project, under the GPL2 license, and the source code is available at http://forge.abcd.harvard.edu/gf/project/screensaver/.

The project home page is https://wiki.med.harvard.edu/ICCBL/Screensaver/.
