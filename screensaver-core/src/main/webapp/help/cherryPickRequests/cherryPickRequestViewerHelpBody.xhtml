The Cherry Pick Request Viewer allows screeners and administrators to view the status of a screener's Cherry Pick Request. It also enables administrators to manage the physical process of creating cherry pick plates for follow-up screenings, as requested by screeners. Please note that the Cherry Pick Request Viewer is currently only functional for RNAi screens.

The Cherry Pick Request Viewer page is divided into five sections that you can collapse or expand by clicking on their headers.

The five sections are:

  • Screen Summary
  • Cherry Pick Request Details
  • Screener Cherry Picks
  • Lab Cherry Picks
  • Cherry Pick Plates
Click on a section name in the above list for help on that particular section.
