You can easily look up reagents (small molecules, siRNAs, etc.) in the ICCB-L libraries collection by clicking on the "Find Reagents" link in the left menu pane, which opens the "Reagent Finder" page. In the input field, type (or paste from the clipboard) the identifier for one or more reagents. The Reagent Finder page contains further instructions explaining how the reagent identifiers should be specified.
