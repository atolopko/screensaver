These are the specifications for the format of RNAi Libraries Excel Files. Screensaver requires the files to be in this format in order to load them properly.

The specifications are a little strict, and there are multiple reasons for this. For one, it is a lot easier to parse files that are in a standard format. Second, it should be relatively trivial to modify files that diverge from this format so that they conform to specifications. Third, it should be beneficial for other reasons if all the RNAi library files are in the same format. (For instance, if we ever need to write a Perl script that takes an RNAi library file as an input.)

Excel file name

It is recommended but not required that the names of the Excel files matched either the name, or the short name, of the Library as is found in Screensaver.

It is recommended but not required that an entire Library is stored in a single Excel file.

Excel worksheets

The Excel file may have multiple worksheets, with a portion of the library contents found on each worksheet. The Excel file must not contain any blank worksheets.

It is recommended but not required that the worksheets to have meaningful names (e.g., the plate number).

Column headers

The first row of the worksheet must contain the column headers. Subsequent rows must contain the data. The following table lists the column headers Screensaver tries to parse, an indication of whether they are required or optional, and a list of synonyms for each header name. Each sheet must contain exactly one column for each required column header - no duplicates are allowed. Optional column headers must contain either zero or one matching columns. Column names are case-insensitive.

It's very easy to add a new synonym for a required column, but we should try to be conservative about it, especially after the initial implementation is complete, since adding new synonyms can make a previously valid Excel file no longer valid. (This can happen if an older worksheet has a second column that matches the new synonym. In this case, the worksheet would now have a duplicate column header.)

Data rows

If all the parsed columns have empty values, then the row is ignored. If the Plate and Well columns are the only parsed column with non-empty values, Screensaver assumes that the well is a control well. Otherwise, every column marked in the table below as "is required for experimental well?" must be contain a non-empty value matching the specified format.
